# Portfolio Websites

React + TypeScript + Vite, with GSAP and Motion (Framer Motion) preconfigured for animation.

Bootstrapped from [react-starter](https://github.com/stamajo-ui/react-starter), a reusable personal template.

## Stack

- [React](https://react.dev/) 19 + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vite.dev/) for dev server and build
- [GSAP](https://gsap.com/) for timeline/scroll animation
- [Motion](https://motion.dev/) (formerly Framer Motion) for React component animation

## Scripts

- `npm run dev` — start the dev server
- `npm run build` — type-check and build for production
- `npm run preview` — preview the production build
- `npm run lint` — run oxlint
